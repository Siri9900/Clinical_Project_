# End-to-End Clinical SAS Project

## Overview
This project demonstrates SDTM, ADaM, and TLF programming using mock clinical trial data.

## Study Design (Mock)
- Phase II
- Randomized, Active-controlled
- Parallel group

## SDTM Domains Developed
DM, CM, AE, EX, VS, LB, QS, SV

## ADaM Datasets Developed
ADSL, ADAE, ADLB, ADVS, ADQS

## Key Derivations
- TRTEMFL (Treatment Emergent Flag)
- AESTDY (Study Day)
- Overall Treatment Arm
- Subject-Level AE Counts
- Time to First TEAE

## Tables, Listings, Figures
- Demog Table
- Disposition Flow
- AE Summary Table
- AE Listing
- Baseline and Post-Baseline Efficacy Summary
- Kaplan-Meier Time to Event Plot

**** Validation

## Programming Techniques Used
- PROC SQL
- PROC FREQ
- PROC REPORT
- PROC TRANSPOSE
- PROC LIFETEST/ SGPLOT
- Subject-level deduplication logic

## Tools
- SAS 9.4
- Git Version Control
